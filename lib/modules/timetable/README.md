# Timetable Module

## Overview
Independent module for live timetable management with real-time updates.

## API Endpoints (to be implemented)
- `GET /api/timetable` - Get timetable
- `POST /api/timetable/update` - Update timetable
- `GET /api/timetable/changes` - Get recent changes (WebSocket)

## Features
- Live timetable updates
- Room assignment management
- Schedule change notifications
- WebSocket for real-time sync
