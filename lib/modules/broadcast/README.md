# Broadcast Module

## Overview
Independent module for sending announcements via in-app notifications and email.

## API Endpoints (to be implemented)
- `POST /api/broadcast/send` - Send announcement
- `GET /api/broadcast/list` - Get announcement history
- `POST /api/broadcast/email` - Send email broadcast

## Features
- In-app notifications
- Email integration
- Target audience selection (all, students, teachers, specific groups)
- Scheduled broadcasts
