# Library Module

## Overview
Independent module for library services including book issuance, returns, and due tracking.

## API Endpoints (to be implemented)
- `POST /api/library/issue` - Issue book
- `POST /api/library/return` - Return book
- `GET /api/library/books` - Get book list
- `GET /api/library/due` - Get due counter

## Features
- Book issuance and returns
- Automated due date tracking
- Reminder notifications
- Fine management
