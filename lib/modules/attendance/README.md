# Attendance Module

## Overview
Independent module for QR-based attendance tracking and analytics.

## API Endpoints (to be implemented)
- `POST /api/attendance/mark` - Mark attendance via QR
- `GET /api/attendance/records` - Get attendance records
- `GET /api/attendance/analytics` - Get analytics data

## Features
- QR code generation and scanning
- Real-time attendance tracking
- Analytics and reporting
- Role-based access (students view, teachers mark, admins manage)
