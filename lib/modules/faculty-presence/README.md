# Faculty Presence Module

## Overview
Independent module for tracking faculty cabin status (Available/Busy/Offline).

## API Endpoints (to be implemented)
- `GET /api/faculty/status` - Get faculty status
- `POST /api/faculty/status/update` - Update status
- `GET /api/faculty/status/live` - Live status updates (WebSocket)

## Features
- Real-time status tracking
- Status indicators (Available/Busy/Offline)
- WebSocket for live updates
- Status history
