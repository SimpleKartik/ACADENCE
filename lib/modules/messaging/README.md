# Messaging Module

## Overview
Independent module for student-teacher communication.

## API Endpoints (to be implemented)
- `POST /api/messaging/send` - Send message
- `GET /api/messaging/conversations` - Get conversations
- `GET /api/messaging/messages/:id` - Get messages (WebSocket)

## Features
- Direct messaging between students and teachers
- Real-time chat via WebSocket
- Message history
- File attachments support
